# selexor
**selexor** is a Python 3 module containing algorithms for feature selection and extraction.

# Installation
#### Dependencies
selexor requires:
- Python >= 3.6
- joblib >= 0.16.0
- nptyping >= 1.3.0
- numpy >= 1.19.1
- scikit-learn >= 0.23.2
- scipy >= 1.5.2
- threadpoolctl >= 2.1.0
- typish >= 1.7.0

#### User installation
The easiest way to install selexor is using ``pip``.\
``pip install selexor``
