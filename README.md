# selexor
**selexor** is a Python 3 module containing algorithms for feature selection and extraction.

# Installation
#### Dependencies
selexor requires:
- Python >= 3.6
- joblib==1.0.1
- numpy==1.20.1
- scikit-learn==0.24.1
- scipy==1.6.0
- threadpoolctl==2.1.0

#### User installation
Clone GitHub repository:\
``git clone https://github.com/SN4KEBYTE/selexor``  
``cd selexor``

Run setup.py script:\
``python setup.py install``
